Alke Wallet 💰

Alke Wallet es una aplicación de billetera digital diseñada para ofrecer una solución segura, dinámica y fácil de usar para la gestión de activos financieros personales. Este proyecto forma parte del Módulo de Fundamentos de Desarrollo Front-end de mi Bootcamp Python Fullstack.


🚀 Descripción del Proyecto

El objetivo principal fue desarrollar una interfaz funcional que permita a los usuarios administrar su dinero de forma digital. La aplicación simula el flujo completo de una fintech, desde el acceso seguro hasta la gestión de transacciones en tiempo real.
Características Principales:

    Gestión de Acceso: Sistema de registro e inicio de sesión con validación de credenciales.

    Panel Financiero: Visualización de saldo disponible mediante tarjetas de resumen dinámicas.

    Transacciones Dinámicas: Simulación de depósitos, retiros y envío de fondos a contactos agendados.

    Historial de Movimientos: Registro detallado y visualización de todas las transacciones realizadas.


🛠️ Stack Tecnológico

Para este desarrollo se utilizaron herramientas que garantizan una interfaz responsiva y una manipulación eficiente del DOM:

    HTML5 & CSS3: Estructura semántica y diseño personalizado con un enfoque en UI/UX para fintech.

    JavaScript (ES6): Lógica para la gestión de saldo, eventos de usuario y validaciones.

    Bootstrap 5: Framework utilizado para agilizar el diseño responsivo, componentes visuales (modales, cards) y la barra de navegación.

    jQuery: Implementación de efectos visuales, animaciones y optimización del manejo de eventos en formularios.

📂 Estructura del Proyecto

La navegación se divide en pantallas esenciales para un flujo de usuario coherente:

    login.html: Acceso seguro a la plataforma.

    menu.html: Dashboard principal con resumen de saldo.

    deposit.html: Interfaz para realizar depósitos y actualizar saldo dinámicamente.

    sendmoney.html: Formulario de transferencias con validación y lista de contactos.

    transactions.html: Visualización optimizada del historial de movimientos.
